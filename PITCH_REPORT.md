# 銷售報表系統項目簡報

## 1. 項目背景

公司每天以手寫發貨單記錄不同客戶的訂單，包括貨品、售出数量及售價。職員再把資料輸入 Excel。

現有問題：

- 每張訂單分開儲存，無法快速查看全日總數；
- 月底需要人手整理大量檔案；
- 難以比較客戶、貨品及分類表現；
- 檔名、日期或資料輸入錯誤不容易發現；
- 管理層缺乏即時、清晰的營運數據。

## 2. 解決方案

建立一套安裝於公司 Windows 11 電腦的「銷售報表系統」。

系統讀取每日所有訂單 Excel，自動驗證、合併及產生每日與每月報表。

```text
手寫發貨單
→ 職員輸入每張訂單 Excel
→ 訂單檔案存入當日資料夾
→ 系統讀取所有訂單
→ 自動產生每日及每月報表
```

## 3. 訂單檔案標準

每張 Excel 代表一個客戶訂單。

檔名格式：

```text
YYYYMMDD_客戶名_序號.xlsx
```

例如：

```text
20260619_海景酒家_001.xlsx
```

訂單模板包括：

- 客戶名稱；
- 日期；
- 發貨單編號；
- 貨品分類：牛、豬、魚、羊、雜貨；
- 貨品及部位；
- 售出数量；
- 平均買入價；
- 平價切或精修切售價；
- 訂單總数量及總收入。

## 4. 非技術人員操作流程

桌面捷徑：

```text
Sales Report System
```

每日操作：

1. 按「打開今日訂單資料夾」。
2. 把當日訂單 Excel 放入資料夾。
3. 按「產生今日報表」。
4. 按「打開今日報表」。

月報操作：

1. 按「產生本月月報」。
2. 按「打開本月月報」。

無需使用 PowerShell、程式碼或複雜 Excel 公式。

## 5. 自動化功能

- 自動選擇今日訂單資料夾；
- 自動建立今日及未來七天的日期資料夾；
- 自動命名每日報表：`每日報表_YYYYMMDD.xlsx`；
- 自動命名月報表：`月報表_YYYYMM.xlsx`；
- 保留舊報表，不會因新一天而被覆蓋；
- 記住訂單及報表預設位置；
- 如選錯日期資料夾，顯示確認警告；
- 如報表正在 Excel 開啟，顯示無法更新提示。

## 6. 資料驗證

系統會檢查：

- 檔名是否符合標準；
- 檔名日期與 Excel 日期是否一致；
- Excel 日期與日期資料夾是否一致；
- 客戶名稱、日期及發貨單編號是否缺漏；
- 售出数量是否有效；
- 是否缺少售價；
- 平價切及精修切售價是否同時填寫；
- 是否存在無貨品名稱但有数量或價錢的資料；
- 是否沒有任何已售出貨品。

問題會集中顯示於報表的「問題」頁，方便職員更正。

## 7. 每日報表

每日報表提供：

- 總售出数量；
- 總收入；
- 按客戶統計；
- 按貨品統計；
- 按分類統計；
- 完整銷售明細；
- 輸入問題清單。

## 8. 月報表分析

月報表已精簡為八個主要頁面：

```text
總覽
月度比較
客戶分析
貨品分析
分類分析
每日走勢
明細
問題
```

### 總覽

- 訂單數量；
- 客戶數量；
- 總售出数量；
- 總收入；
- 平均每單收入；
- 平均售價；
- 問題數量。

### 客戶分析

- 各客戶收入及購買数量；
- 客戶收入佔比；
- 十大客戶；
- 本月與上月客戶表現比較。

### 貨品分析

- 各貨品售出数量及收入；
- 平均售價；
- 十大貨品；
- 貨品需求升跌；
- 本月與上月貨品比較。

### 分類分析

- 牛、豬、魚、羊及雜貨的数量與收入；
- 分類收入佔比；
- 本月與上月分類表現比較。

### 每日走勢

- 每日訂單數；
- 每日售出数量；
- 每日收入。

## 9. 圖表

月報表自動產生五個 Excel 圖表：

- 分類收入佔比；
- 每日收入走勢；
- 十大客戶收入；
- 十大貨品收入；
- 本月與上月比較。

所有主要數據亦以可篩選、排序的 Excel 表格顯示。

## 10. 商業價值

- 減少每日及月底人手整合工作；
- 降低輸入及日期分類錯誤；
- 管理層更快掌握收入、数量及客戶表現；
- 找出重要客戶及高需求貨品；
- 監察價格及收入變化；
- 為日後採購預測及庫存管理建立可靠數據基礎；
- 保留公司原有 Excel 工作方式，減少職員培訓成本。

## 13. 60 秒介紹稿

公司現時每天使用手寫發貨單，再由職員輸入個別 Excel。問題是所有訂單分散，管理層無法快速查看每日總收入、貨品数量及整月趨勢。

這套銷售報表系統保留公司熟悉的 Excel 流程。職員只需要把每張訂單存入當日資料夾，再按一個按鈕。系統便會自動讀取、檢查及合併所有訂單，產生每日報表。

月底時，系統會自動整合整月資料，顯示客戶、貨品、分類、收入、数量、價格及上月比較，並自動產生圖表。

整套系統以非技術職員為設計對象，操作簡單，同時為公司建立可持續使用的銷售數據基礎。
